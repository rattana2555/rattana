## 🔐 Login
- **วิธี:** พิมพ์ **รหัสพนักงาน** (Column C ใน Sheet `User`)
- **เช็คสิทธิ์:** User Role ใน Column P ต้องมีค่า (ไม่ว่าง)
- **Session:** จำไว้ **7 วัน** ไม่ต้อง login ซ้ำ
- **ไม่ต้อง** Google OAuth / Cloud Console ใดๆ ทั้งนั้น ✅
